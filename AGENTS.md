## Architecture

- **nomos-api**: Python 3.12 FastAPI (19 routers, 49+ endpoints incl. STH + inclusion-proof)
- **nomos-console**: Next.js 15 / React 19 (20 pages)
- **nomos-plugin**: TypeScript OpenClaw gateway plugin (11 hooks)
- **Data**: PostgreSQL 16 + pgvector, Valkey (cache), HashiCorp Vault (secrets)

## Key Commands

### CLI (Local)
```bash
nomos hire --name "Mani" --role external-secretary --output-dir ./data/agents/mani
nomos verify --agent-dir ./data/agents/mani
nomos gate --agent-dir ./data/agents/mani
nomos fleet --agents-dir ./data/agents
```

### API (Remote)
```bash

```

## Development

### Lint & Typecheck
```bash
# Python
ruff check nomos-cli nomos-api
ruff format --check nomos-cli nomos-api

# TypeScript
cd nomos-plugin && npm run typecheck
cd nomos-console && npm run typecheck
```

### Tests
```bash
# Python
cd nomos-cli && uv run pytest
cd nomos-api && PYTHONPATH="../nomos-cli" uv run pytest

# TypeScript
cd nomos-plugin && npm test
cd nomos-console && npm test
```

### Contract Check
```bash
python scripts/export-schemas.py > schemas.json
npx tsx scripts/check-contracts.ts
```

## CI Pipeline

1. **Lint**: `ruff` (Python), `tsc` (TypeScript)
2. **Unit Tests**: `pytest` (Python), `vitest` (TypeScript)
3. **Contract Check**: `schemas.py` vs `types.ts`
4. **Quality Gate**: No TODOs, no internal IPs, no secrets
5. **Docker Build**: `docker compose build`

## Important Notes

- **Secrets**: Never commit `.env`, `vault/`, or API keys
- **Audit Trail**: SHA-256 hash chain in `audit/` â€” verify with `nomos audit --verify`
- **Compliance Gate**: Blocks deployment until 5 EU AI Act documents are signed
- **PII Filter**: Real-time redaction of personal data (DSGVO Art. 6)
- **Budget Control**: Per-agent cost limits with fail-closed behavior

## Compliance Documents

Generated by `nomos gate`:
- DPIA (Data Protection Impact Assessment)
- Art. 30 (Records of processing activities)
- Art. 50 (Transparency obligations)
- Art. 14 (Right to information)
- Art. 12 (Transparent information)
